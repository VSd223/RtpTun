## ✨ Возможности

- 🔒 **Надёжное шифрование** — трафик защищён на уровне транспортного протокола
- ⚡ **Высокая скорость** — минимальные задержки благодаря нативному ядру
- 🤖 **Авто-капча** — автоматическое прохождение VK Smart Captcha без вашего участия
- 🌐 **Несколько серверов** — добавляйте и переключайте профили в один тап
- 🔁 **Авто-переподключение** — сеть пропала? Туннель восстановится сам
- 📡 **Quick Settings** — кнопка включения прямо в шторке уведомлений
- 🎨 **Темы оформления** — тёмная, светлая, синяя, коричневая + динамические цвета
- 📋 **Логи сессий** — просмотр и экспорт в один тап

## ❓ Часто задаваемые вопросы

<details>
<summary><b>Приложение просит подозрительные разрешения?</b></summary>

VPN-приложению нужны права на VPN-сервис, фоновую работу и уведомления — это стандарт для любого VPN на Android. Ничего лишнего.

</details>

<details>
<summary><b>Что такое капча и почему она решается автоматически?</b></summary>

Сервер использует инфраструктуру VK. При первом соединении или после долгого перерыва появляется капча. Приложение проходит её самостоятельно — вам ничего делать не нужно.

</details>

<details>
<summary><b>VPN отключается при смене Wi-Fi / мобильного интернета?</b></summary>

Нет. Приложение автоматически отслеживает смену сети и восстанавливает соединение.

</details>
