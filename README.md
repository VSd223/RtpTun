## 💎 О проекте

**RtpTun** — это высокотехнологичный VPN-клиент нового поколения, разработанный специально для работы в условиях жестких сетевых ограничений. Проект ориентирован на максимальную маскировку трафика и стабильность соединения на мобильных устройствах. 

Это **Proprietary** (закрытый) проект. Весь интерфейс приложения написан с нуля, а внутреннее сетевое ядро глубоко модифицировано для обеспечения эксклюзивных функций и поддержки уникальных методов шифрования.

---

## ⚡ Технологии и методы работы

RtpTun использует уникальный метод туннелирования, который делает ваш трафик невидимым для систем глубокого анализа пакетов (DPI):

*   **RTP Masking (Audio Stream Mimicry):** В отличие от обычных VPN, RtpTun упаковывает каждый пакет данных в RTP-контейнер (Real-time Transport Protocol). Для любого наблюдателя и провайдера ваш трафик выглядит как обычный голосовой звонок (WebRTC Opus), что исключает возможность шейпинга скорости.
*   **Dual-Layer AEAD Encryption:** Применение алгоритма ChaCha20-Poly1305 обеспечивает двухслойную защиту данных. Каждый байт зашифрован уникальным ключом, производным от вашего пароля через функцию деривации HKDF.
*   **Multi-Protocol Discovery:** Ядро версии 1.2.0 способно в реальном времени определять тип протокола на стороне сервера и автоматически переключаться между новейшим форматом **RtpTun v1** и поддержкой устаревших узлов.
*   **Adaptive Chunking:** Диспетчер трафика автоматически группирует пакеты для минимизации потерь и предотвращения задержек, характерных для мобильных сетей.

## ✨ Основной функционал

- 🔒 **RtpTun Secure Wrap** — уникальное шифрование, ключ от которого невозможно найти внутри APK.
- 🤖 **Neural Captcha Solver** — каскадная система прохождения проверок: фоновое решение через автоматику и умное полноэкранное окно в случае необходимости.
- 🚀 **Hardware Acceleration** — навигация и графические эффекты используют ресурсы видеочипа (GPU), обеспечивая плавность интерфейса.
- 🌐 **Location Management** — поддержка неограниченного числа серверов с индикацией реальной задержки (Ping).
- 🔁 **Active Network Watchdog** — мгновенное восстановление туннеля при переходе между Wi-Fi и LTE.
- 📡 **Quick Access** — виджет управления на рабочий стол и кнопка в шторке уведомлений Android.
- 🎨 **Adaptive Themes** — глубокая настройка внешнего вида под любые предпочтения.

## 📲 Установка

1. Перейдите в раздел **[Последние релизы](../../releases)**.
2. Загрузите APK-файл под вашу архитектуру (рекомендуется универсальная или ARM64).
3. В настройках Android разрешите установку приложений из сторонних источников.
4. После запуска предоставьте приложению права на использование VPN-сервиса и отображение поверх окон.
